# Bujit (work in progress)

A budgeting and expense tracking app made with React.

![demo](/media/demo.gif) ![demo2](/media/demo2.gif)

## Implemented features

- Create/delete account
- Add/remove money
- Edit account info
- Transfer money between accounts
